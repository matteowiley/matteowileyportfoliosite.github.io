Flipster

Flipster is based on the classic game of memory with a social media twist. A player logs in with Instagram and starts a new game. Then waits for an opponent to challenge your memory using both players photos from Instagram. This application is built 100% in JavaScript using Sails. While the goal was to utilize the real time WebSockets built into Sails many other features became useful during the process. Check out the code!

Wiley Pecan Farm

Designed an e-commerce platform for my family's Pecan Orchard. Main functions includes ordering fresh pecans from Tow, Texas and also reaches prospective and established customers by e-mail (for example, with newsletters). Technologies used are Angular, Ruby on Rails, JavaScript, HTML5, and CSS. Check out the code!
